# 📊 Contador de Cliques

Projeto desenvolvido com foco em prática de lógica de programação, manipulação do DOM e aplicação de conceitos de Qualidade de Software (QA).

---

## 🚀 Demonstração

🔗 Acesse o projeto online:
https://jaquelinesanches.github.io/contador-de-cliques/

📁 Repositório:
https://github.com/JaquelineSanches/contador-de-cliques

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Git e GitHub

---

## ⚙️ Funcionalidades

- ➕ Incremento e decremento do contador
- 🔢 Ajuste do valor em diferentes intervalos (+1, +5, +10, -1, -5, -10)
- 🔄 Reset do contador
- 🌙 Alternância entre modo claro e escuro (dark mode)
- 💾 Persistência de dados com LocalStorage
  - Valor do contador
  - Histórico de ações
  - Preferência de tema

- 📜 Histórico das últimas ações realizadas (limitado a 5 registros)
- 🎯 Limite mínimo (0) e máximo (999) para o contador
- 🎨 Feedback visual com animação e alteração de cores
- 📱 Layout responsivo
